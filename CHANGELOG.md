# 更新日志

记录这个项目的每次更新。版本号用 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.3.0] - 2026-06-05

### 重构 / 新增
- ♻️ 把日期、连续天数、最长记录、完成率统计、数据规范化等纯逻辑抽到 `habit-logic.js`（与界面解耦，并改为可注入"今天"以便确定性测试）
- 🧪 新增 13 个单元测试（Node 内置测试框架，零依赖），覆盖连续/断点/跨月/窗口统计/规范化等场景
- 🤖 GitHub Actions CI：每次 push 在 Node 18/20/22 上自动跑测试

## [1.2.1] - 2026-06-05

### 修复 / 优化
- 🛡 健壮性：导入与加载时规范化数据，缺字段自动补齐、垃圾项丢弃；localStorage 损坏也不再崩溃
- ♿ 可访问性：上移/下移/删除按钮加 `aria-label`，补 `description` / `color-scheme` 元信息

## [1.2.0] - 2026-06-05

### 新增
- 🎉 连续打卡命中里程碑（3/7/14/21/30/50/100 天）时弹彩蛋鼓励
- 📝 每天可写一句备注，热力图悬停可查看
- 🏷 GitHub 主题标签（heatmap、productivity 等）方便被搜到
- 🏅 README 加 MIT + 零依赖徽章

## [1.1.0] - 2026-06-04

- ✏️ 双击习惯名称即可重命名
- ↑↓ 上移 / 下移按钮，可调整习惯顺序
- 💾 导出 / 导入 JSON 备份（换浏览器或电脑也不丢数据）

- 更新截图，展示新功能
